1️⃣ What is the difference between var, let, and const?

Ans: Let is a variable value. It is changeable. We can assign a value later in which we defined as let even if a value is assigned before.

Const is a constant value. We cannot change the value of this once we define the value. It is constant forever.

Var is similar to let but with some issues. It is function scoped. We can get it in whole function even if we define it inside if else condition.we can reassign a same value twice or as much as we want in var but we cannot do it in let.

2️⃣ What is the spread operator (...)?

Ans: It is an operator that breaks array or object into pieces and spreads them out. We can work with single elements of array o objects using spread operator.Basically it works like looping an array or object but in only one line of code.We can add or remove elements easily in array or object using spread operator.

3️⃣ What is the difference between map(), filter(), and forEach()?

Ans: forEach() is used only to loop through something.It does not return anything.

map() takes an array, transforms it into a newly created array and the root array remains unchanged.

filter() mainly filters thing based on conditions. If condition is fulfilled then it returns the thing otherwise doesn't return anything.

4️⃣ What is an arrow function?

Ans: Arrow function is a way to write function shortly.A general function has it's own "this" but arrow function doesn't have it.also it cannot be used as a constructor.

5️⃣ What are template literals?

Ans: template literal is a modern way of ES6 to write strings. We use backticks instead of single cote or double cote. The main benefit we get is we can use multi line string with this and also, can use dynamic variables and write js expression inside it.
